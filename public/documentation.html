MyFlix API Documentation

Introduction

MyFlix is an API that provides information about movies and allows users to register, update their username, add and remove movies from their favorites list, and deregister. The API provides the following endpoints:

GET /movies/:title

Returns information about a single movie by its title.

  • Endpoint URL: /movies/:title
  • Method: GET
  • Query Parameters:
    • title: the title of the movie to retrieve (string)
  • Response Format: JSON
  • Response Body Example: 
    {
  {
    "Genre": {
        "Name": "Science Fiction",
        "Description": "A genre of film that typically involves futuristic or speculative elements, often exploring technological advances, scientific theories, or alternate realities. "
    },
    "Actors": [],
    "_id": "6406b68286d26a571737a40f",
    "Name": "Blade Runner",
    "Description": "A retired cop is brought back to hunt down a group of rogue replicants, androids that look like humans, in a dystopian Los Angeles of the future.",
    "Director": {
        "Name": "Ridley Scott",
        "Bio": " An English film director and producer, Ridley Scott is known for his work in science fiction, including classics such as \"Blade Runner\" and \"Alien\".",
        "Birth": "11/29/1947",
        "Death": "N/A"
    },
    "ImagePath": "blade.png",
    "Featured": true
}
}

GET /movies/genre/:genreName

Returns a list of all movies in a specific genre.

  • Endpoint URL: /movies/genre/:genreName
  • Method: GET
  • Query Parameters:
    • genreName: the name of the genre to retrieve (string)
  • Response Format: JSON
  • Response Body Example: 
    [   {
  {
    "_id": "640635bc86d26a571737a3fb",
    "Name": "Comedy",
    "Description": "Comedy is a genre of film that uses humor and satire to entertain viewers and often tackles serious topics in a lighthearted way."
}
  ]

POST /Users

allows a new user to regiester

  • Endpoint URL: /users
  • Method: POST
  • Response Format: JSON
  • Response Body Example: 
              {
            "Username": "AnjaliSeedhar",
            "Password": "1Direction4Life",
            "Email": "anjpreet@gmail.com",
            "Birthday": "1997-09-04T04:00:00.000Z",
            "FavoriteMovies": [],
            "_id": "6408c4cfbd2615732f33f73d",
            "__v": 0
        }

POST /users/:Username/movies/:MovieID

allow users to add a list of movies to there favourites

  • Endpoint URL: movies/:id/:movieTitle
  • Method: POST
  • Response Format: Plain Text
  • Response Body Example: 
              {
            "_id": "6408c4cfbd2615732f33f73d",
            "Username": "AnjaliSeedhar",
            "Password": "1Direction4Life",
            "Email": "anjpreet@gmail.com",
            "Birthday": "1997-09-04T04:00:00.000Z",
            "FavoriteMovies": [
                "6406bee086d26a571737a41e"
            ],
            "__v": 0
        }

UPDATE /users/:Username/movies/:MovieID

Allow users to remove a movie from there list

  • Endpoint URL: /movies/:id/:movieTitle
  • Method: DELETE
  • Response Format: Plain Text
  • Response Body Example: 
              {
            "_id": "6408c4cfbd2615732f33f73d",
            "Username": "AnjaliSeedhar",
            "Password": "1Direction4Life",
            "Email": "anjpreet@gmail.com",
            "Birthday": "1997-09-04T04:00:00.000Z",
            "FavoriteMovies": [],
            "__v": 0
        }

DELETE /users/:Username

Allow users to deregister

  • Endpoint URL: /users/:id
  • Method: DELETE
  • Response Format: Plain Text
  • Response Body Example: 
              "6408c4cfbd2615732f33f73d" has been deleted
